# Ecommerce Management — Frontend

Interface administrativa para o Ecommerce Management.

Resumo

- Aplicação frontend em React + Vite + TypeScript.
- Painel administrativo para gerir produtos, vendas, compras, inventário e usuários.

Pré-requisitos

- Node.js 18+ e yarn/npm

Instalação e execução

1. Instalar dependências:

```bash
npm install
```

2. Rodar em modo de desenvolvimento:

```bash
npm run dev
```

Build para produção

```bash
npm run build
```

Notas

- Ajuste a URL da API em `src/services/api.ts` para apontar para o backend local ou remoto.
